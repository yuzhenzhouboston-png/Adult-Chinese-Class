# Adult Chinese Class Review Center

This is a small bilingual review website for an adult Chinese class.

Students can use it to:

- Review class slides
- Complete matching and sentence-building homework
- Watch listening scenario videos
- Practice scenario dialogue

The page supports Chinese and English interface text on the same page.

## Class Access Code

The current access code is configured in `site-config.js`.

## GitHub Pages

Publish this repository with GitHub Pages from the `main` branch and root folder.
